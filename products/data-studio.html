DataStudio

Fully managed enterprise grade web crawler that transforms websites into datasets.

Crawl

InvanaBot lets you setup the crawling pipelines with traversals paths, and data extraction features. You can extract standard data like SEO’s metadata or all the paragraphs, links or tables in the page. You define custom data extraction config that let’s you extract specific data from the page.

Transform

Write your own logics to covert the crawled data into a useful data schema that would serve your goals.

Augment your data by adding more data on top of existing data. Apply pre-build machine learning models on top of data like description, or review texts to add sentiments or category data on top of your data.

Index

Your crawled + transformed data is always ready for production use. The data is indexed into a search engine and is available to be served as an API , that you can make queries with Elasticsearch's DSL .
