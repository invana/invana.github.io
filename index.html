Model, Build and Explore Knowledge Graphs with Invana

We help people explore, analyse and find insights from data using the power of Knowledge Graphs.

Learn more Get started

Features

Connect to any Apache TinkerPop enabled graph database, run queries, build and organise insights.

Model data into Unified Schema

  • Establish relationships between entities from multiple sources
  • Design and organise data into a unified schema for your knowledge graphs

Instant GraphQL APIs for Knowledge Graphs

  • Consume knowledge graphs instantly as GraphQL API
  • Integrate knowledge graphs to your applications using GraphQL API

Import any data into knowledge graphs with ease

  • Python and GraphQL API for programmatic access to knowledge graphs
  • Import data from database or any third-party apps integrating with the APIs

Visualise data as Interactive Directed Graphs

  • Visualise entities and relationships as interactive directed graphs
  • Load graphs almost with no queries for most cases
  • Extend your curiosity interactively expanding in and out relationships of an entity

OLTP and OLAP support with Apache TinkerPop.

  • Supports Online transaction processing (OLTP) and Online analytical processing (OLAP) with Apache TinkerPop's Gremlin
  • Query knowledge graphs with Apache TinkerPop's Gremlin

Switch graph database backend any-time

  • Invana supports Apache TinkerPop enabled graph databases
  • Need fixes on the element ID resolution for new database support and any graph database should work with Invana

The Technology

Invana makes modeling, building and exploring knowledge graphs easy using the following products.

Invana Engine

GraphQL API for building and querying knowledge graphs.

Key features include:

  • Works using Apache TinkerPop™ graph computing framework.
  • Python and GraphQL API.
  • Currently supports JanusGraph only, more databases support shall be added soon.

Invana Studio

Visualise and traverse through knowledge graphs.

Key features include:

  • Query with Apache TinkerPop's Gremlin
  • Visualise as force-directed Graphs and Tables
  • Traverse through data extending in and out data.
  • Save your graph as Image
