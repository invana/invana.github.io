Invana is an open-source distributed systems orchestration engine aiming to allow you run container-based microservices that range from running analytics to machine learning models, on top of data streams, giving you a production-ready Knowledge and Data APIs in near realtime.

Checkout technology

What we offer you to Build Knowledge Graphs at Scale

Streams

Define and use communication channels for your data flow. Add as many subscribers as you require to scale up your task execution in parallel.

Functions

Execute Python functions on top of your data streams, giving abilities to clean, manipulate or enrich event data.

Data Management

Use databases like PostgreSQL, MongoDB, Elasticsearch as data stores; more database yet to be supported.

Event Timeline

Keeps a track of all the events happened and gives you a timeline of the events. Giving you ability to audit or migrate your data to new Schema.

Topology Editor

Build Schema-driven knowledge graph with data from multiple streams. Never worry about migrations, as we can re-generate knowledge graph using event timeline.

Distributed Resources Manager

Create container-based microservices, deploy, schedule and Monitor services across different cloud platforms like GCP, AWS. You can deploy resources in the regions that best suite you.

OLAP & OLTP Support

Run queries, perform analytics, computations on top your data using graph traversal language `Gremlin` developed by Apache's Tinkerpop of Apache Software Foundation.

Graph Visualiser

Web Interface to access your data as well connected Graph data.
